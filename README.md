# 🟢 PORTFOLIO.OS — Personal Academic & Dev Workspace

A fully customizable, personal portfolio designed as an interactive operating system interface. This platform centralizes my academic background, technical skills, projects, and future ambitions.

---

## 🚀 Key Features

* **Multilingual Core:** Built-in language switcher supporting French, English, Español, Deutsch, and Русский.
* **Dynamic Theme Engine:** Switch interface styles on the fly with custom themes including Matrix, Cyberpunk, Business, Retro, and Minimal, alongside a global Dark/Light mode toggle.
* **Modular Multi-page Architecture:** Organized tabs separating current activities (`actuel.html`), future goals (`futur.html`), projects showcase (`projets.html`), and certifications.
